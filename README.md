# StockPulse

Stock Market Portfolio App built with React + Vite and Supabase.

## Tech Stack

- Frontend: React + Vite + Tailwind CSS
- Backend: Supabase (PostgreSQL + Auth + Realtime)
- Stock Data: Finnhub API

## Getting Started

### Prerequisites
- Node.js 18+
- A Supabase project
